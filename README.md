# Personal_CRM
My Personal CRM System

## Dir Structure
```
personal_crm/
│
├── app.py                # The main web server and logic
├── database.py           # Script to initialize the robust database
├── templates/            # HTML files for the UI
│   ├── base.html         # Main layout (navbar, footer)
│   ├── index.html        # Dashboard
│   ├── people.html       # People list and add form
│   ├── places.html       # Places list and add form
│   └── journal.html      # Events list
```
